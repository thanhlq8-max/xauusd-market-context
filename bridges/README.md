# bridges/

Reserved path for future monitor-only alert bridges.

Allowed future bridge outputs:

- `TRACK_RECLAIM`
- `TRACK_ACCEPT`
- `TRACK_DELIVERY`
- `D_ACTIVE`
- `D_STALL`
- `D_COUNTERFLOW`
- `D_FAILED`
- `TARGET_HIT`

Forbidden without a separate approved execution spec:

- order placement;
- position sizing;
- stop-loss/take-profit instruction;
- account-risk automation;
- broker execution;
- guaranteed-profit or signal claims.
